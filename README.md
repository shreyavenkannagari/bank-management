# bank-management
**BLOCKCHAIN TECHNOLOGY FOR PROTECTING BANKING TRANSACTIONS WITHOUT USING TOKENS**
Developed a secure blockchain-based system to safeguard banking transactions, ensuring data integrity and transparency. 
Enhanced security by eliminating token-based authentication and implementing decentralized validation mechanisms.
**Technologies Used:**
Utilized Java/J2EE (JSP, Servlet), JavaScript, HTML, CSS for frontend development, and MySQL as the backend,
ensuring a robust and secure system.
